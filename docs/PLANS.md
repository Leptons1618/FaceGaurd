# Project Plans

## Goals
- High-performance, on-device face detection and recognition
- Edge AI optimization (NVIDIA Jetson Orin Nano)
- Robust privacy and security (offline-first)
- Modular, extensible Rust architecture
- Modern UI (Dioxus/Tauri)

## Milestones
- [x] Initial project setup
- [x] Architecture and feature documentation
- [ ] Core algorithm implementation (camera, detection, recognition, tracking)
- [ ] UI integration (Dioxus frontend, Tauri backend)
- [ ] WASM/web build pipeline
- [ ] Edge device optimization (Jetson, CUDA/TensorRT)
- [ ] Documentation and testing

## Roadmap
1. Finalize core engine: camera pipeline, face detection, recognition, tracking, event engine
2. Integrate Dioxus UI and Tauri backend
3. Enable WASM/web builds for browser support
4. Optimize for Jetson/CUDA/TensorRT
5. Expand documentation and feature list
6. Release MVP
7. Plan for advanced features (liveness, multi-camera, mobile app)
